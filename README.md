AI Workplace Productivity Assistant

Project Overview

The AI Workplace Productivity Assistant is a web-based application designed to improve workplace productivity using Artificial Intelligence. The application helps users automate repetitive workplace tasks such as writing emails, summarising meeting notes, and planning daily tasks.

The project was built using Lovable AI, GitHub, and ChatGPT.

Features

Smart Email Generator

Generates professional workplace emails in different tones such as formal, informal, and persuasive.

Meeting Notes Summariser

Summarises meeting notes into concise key points, action items, responsibilities, and deadlines.

AI Task Planner

Creates organised task schedules and helps users prioritise important work activities.

Responsible AI Section

Encourages ethical AI usage, warns users about AI inaccuracies, and promotes verification of AI-generated outputs.
